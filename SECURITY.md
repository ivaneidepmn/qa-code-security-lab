# 🔐 Security Policy

## Supported Versions

This laboratory project follows controlled security validation practices.

| Version | Supported |
| ------- | --------- |
| 1.0.x   | ✅ Yes |
| < 1.0   | ❌ No |

## Reporting a Vulnerability

If you identify a potential security vulnerability during experiments or code analysis activities, please report responsibly.

Security observations related to this learning laboratory can be documented through:

- GitHub Issues (for educational discussion)
- Security research notes in project documentation
- Controlled test evidence inside the /docs/evidences directory

This project simulates proactive security engineering practices aligned with DevSecOps culture and risk-based quality strategy.

⚠️ This repository is intended for educational and research purposes only.
